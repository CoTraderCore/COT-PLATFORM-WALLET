# description
```
This contract get 10% from CoTrader managers profit
and then distributes assets

Convert some % of asset to COT and burn (transfer to dead address)
Convert some % of asset to COT and send to stake reserve
Destribute some % (but can't more than 40%) of asset to owner of this contract (CoTrader team)

Note 51% CoTrader token holders can change owner of this contract  
```

# run test
```
1) npm i
2) npm run ganache (in separate console)
3) truffle test
```
